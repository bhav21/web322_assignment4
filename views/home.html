What is HTML5?

This website uses HTML5 - a markup language used for structuring and present what you see on the web. It is the latest version of HTML which supports new markup language functionalities like multimedia, new elements and tags (eg, header, footer, nav, article) along with new APIs. HTML5 was created to improve the WWW experience for both the developers as well as the users. It allows web browsers to function as application platforms, enhancing developers' control over their website's performance. In the case of the end users, some of the main advantages of HTML5 are -

  • users do not require any additional plug-ins for audio or video elements on a page
  • it makes use of offlice caching so that users can have access to webpages even when they are not connected to the network
  • web forms are more enhanced with better txt inputs, search boxes and fields for different purposes
  • it offers geolocation of the user while they browse a site or application

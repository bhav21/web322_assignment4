Working with HTML5 Lists

HTML lists are widely used to represent a list of item. The two kinds of lists are - ordered list <ol>, which renders a numbered list, and an unordered list <ul>, which is rendered as a bulletted list of items in no specific order. Within the <ol> or <ul> tags, we place <li> tags - the <li> element represents one item in the list.

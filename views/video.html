The HTML5 <video> Element

This is a video element, which is used to embed a media player that supports video playback into the document. I have included two sources for the same video - one is a .mp4 file, and the other is a .ogg file. This is a good practice, as some browsers do not support certain file types. If multiple sources are provided, the browser chooses the most suitable one for playback.
